# 🚀 Placement AI Hub – Resume Analyzer + Career Chatbot

A modern FastAPI-based platform where students can:

✅ Upload resume (PDF)  
✅ Get AI-powered resume analysis using **Google Gemini**  
✅ Get job & internship suggestions  
✅ Chat with AI in multiple languages (English, Hindi, Kannada, Tamil, Telugu)  
✅ View example company requirements (CGPA + Skills)  
✅ Login/Signup + Secure student authentication
